Responsive Design Template

A solid starting point for front-end developers building responsive websites. Built by Kevin Thornbloom. Resize this page for a demo.

Why is it awesome?

  1. Multi-level, viewport aware dropdowns
  2. It will automatically create off-canvas mobile navigation based upon your main navigation. Yay for less work!
  3. It contains code examples for several types of responsive elements
  4. The mobile navigation animation uses CSS3 so it's silky smooth.
  5. Unlike other templates/boilerplates, it doesn't force you to use any particular layout method besides an unordered list for the navigation. Code how you want to!
  6. FREE for personal, professional, or other projects. Just don't attempt to resell this as-is

How do I use it?

  1. Download the files from Github
  2. Use the given structure in index.html, but besides that- go nuts!
  3. Use a class of "content" for the container holding your main content to take advantage of pre-written CSS hooks

Examples of responsive elements

Page Images

I personally use classes of ".float-right" and ".float-left" for my page images. These will be placed as they're named, but will resize down smaller if needed.

Tables? Oh yeah.

Courtesy of Zurb, just add a class of "responsive" to your tables and they'll work just fine on mobile.
