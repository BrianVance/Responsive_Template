RWD TEMPLATE

Template for Responsive Website Development

This template is to be used as a starting point for developing responsive web designs. Resize this example on a desktop to see the nav change, or just check it out on your mobile device.
View Project on Github

Features

  • Automatically creates a sidebar style mobile menu
  • Hardware Accelerated CSS transitions used instead of jQuery animations to provide silky smooth effects.
  • iOS 6 is tricked into using Hardware Acceleration. (It is disabled by default for some reason.)
  • Multi-Level Dropdowns supported for both main & mobile navigation.
  • Great response times due to a script that removed a 300ms standard delay on mobile devices.
  • Responsive iFrame code included.
  • Includes jQuery, jQuery Easing, jQuery Transit, & more.

Requirements

  • The responsive portions of this website require a recent version of Firefox, Chrome, Safari, or Opera. IE9+ should be mostly supported.
  • The site must be built fluidly. (does not require horizontal scrollbars).
  • In mobile view, the page content should preferably be longer than the navigation sidebar for best effect.
