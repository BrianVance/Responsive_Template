Responsive Website Template

This template is a starting point for developing responsive web designs. Below are some of the features. It is admittedly for my personal use, but I am making it available to anyone who might make use of it.

Navigation

  • Built in script to populate the mobile navigation based upon the main navigation
  • Down arrows signify nav items with menues
  • Sidebar Menu style allows mobile users to easily access a large amount of pages

Hardware Acceleration

  • Hardware Accelerated CSS transitions used instead of jQuery animations to provide silky smooth effects
  • iOS 6 is tricked into using Hardware Acceleration (it is diabled by default for some reason)
  • "Fastbutton" script to remove a 300ms delay on mobile devices
